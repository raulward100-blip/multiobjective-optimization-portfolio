# Многокритериальная оптимизация инвестиционного портфеля

## Описание проекта
Исследование методов многокритериальной оптимизации на задаче формирования инвестиционного портфеля с тремя критериями:
- **Доходность** (максимизация)
- **Риск** (минимизация)
- **ESG-рейтинг** (максимизация)

## Методы
В работе реализованы и сравнены:
1. **Метод взвешенных сумм** — линейная свёртка критериев
2. **Метод идеальной точки** — минимизация расстояния до идеала
3. **Метод последовательных уступок** — лексикографический подход
4. **Эволюционный алгоритм NSGA-II** — построение аппроксимации фронта Парето

## Результаты
- Получена аппроксимация множества Парето из 200 портфелей
- Проведён анализ чувствительности параметров NSGA-II (размер популяции, число поколений) с использованием IGD-метрики
- Введены количественные показатели «цены» замены портфеля для оценки компромиссов
- Сформулированы рекомендации по выбору метода в зависимости от условий задачи

## Технологии
- Python 3.x
- NumPy
- Matplotlib
- pymoo (библиотека для эволюционной оптимизации)

## Структура репозитория
- `nsga2_portfolio.py` — реализация NSGA-II на Python
- `portfolio_data.csv` — данные по активам (доходность, риск, ESG)
- `pareto_front.csv` — полученные Парето-оптимальные портфели
- `figures/` — визуализация фронта Парето
